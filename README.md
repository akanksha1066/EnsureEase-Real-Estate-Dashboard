# EnsureEase-R# EnsureEase - Real Estate Market Dashboard

An interactive Power BI dashboard developed for the **EnsureEase** B.Tech final year startup project. This dashboard analyzes real estate pricing trends, volume distribution, and BHK configurations across major cities.

## 📊 Key Features
* **Interactive Slicers:** Dynamic filtering by BHK configurations to drill down into specific real estate demands.
* **Pricing Breakdown:** Visualizes the distribution of properties across Budget, Mid-Range, and Luxury tiers using a precise Pie Chart.
* **Geographical Trends:** Comparative analysis of Average Listing Prices across cities (Mumbai, Delhi, Bengaluru, etc.) using a Clustered Bar Chart.
